## 4. 实验结论与分析 (Analysis)
* **同架构蒸馏全面占优**：在 NDS 指标上达到了 **0.2439**，且在卡车 (Truck)、行人 (Pedestrian) 和摩托车 (Motorcycle) 等高频及中频样本上取得了最佳表现。
* **异构蒸馏瓶颈**：由于 Teacher 和 Student 特征维度/架构差异，mAP 暴跌至 0.0387，蒸馏收益几乎为负。
* **长尾/小样本类别失效**：对于 Bicycle (52个) 和 Traffic Cone (39个) 这类小尺寸且样本极少的物体，三种蒸馏策略下的 AP 均为 0，说明当前小模型在 mini 验证集上的感受野或特征提取能力不足以捕获这些目标。
